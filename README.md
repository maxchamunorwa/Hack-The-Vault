# 🔐 Hack The Vault
This is one of my early Python projects as I learn software development.
I built this command-line Python game where you attempt to crack a secure vault code within a limited number of attempts.

## 🚀 Features
- Randomly generated secret number
- Multiple difficulty levels
- Limited attempts to guess the code
- Feedback after each guess
- Replay option

## 🛠️ Built With
- Python

## 🎮 How to Play
1. Run the program
2. Choose a difficulty level
3. Guess the secret number
4. Use the feedback to get closer to the correct code
5. Crack the vault before you run out of attempts!
